# test-happ-vpn-config-v1.0
This is my first mini project. And here I will tell you how to activate this mini subscription via Happ! 
1.To connect this plug, go to the outlet "releases".
2.Download .txt there.
3.It will contain links to all subscription releases! Select the subscription you need and copy it.
4.And then, in the happ application, click on "Insert from the clipboard". Click on it!
5. It's done!

Subscription Details:
There are only about 150 servers in it, which may not be working at any time, so connect to the servers with the best ping.
